## 🔍 Overview

**Kidinnu** is a curated directory of **191+ AI tools**, carefully organized by professional field and use case. Whether you're a developer, designer, marketer, medical professional, content creator, or student — Kidinnu helps you discover the right AI tool for the job in seconds.

### ✨ Key Features

- **191+ AI Tools** — Hand-picked and continuously updated
- **Professional Categorization** — Tools sorted by industry and use case (marketing, programming, medicine, design, productivity, creativity, and more)
- **Fast Search & Filter** — Find exactly what you need instantly
- **Modern UI/UX** — Clean, responsive interface built with shadcn/ui
- **Blazing Fast** — Powered by Vite for near-instant load times
- **Mobile Friendly** — Fully responsive across all devices

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or [bun](https://bun.sh/)

### Installation

```bash
# Clone the repository
git clone https://github.com/3h0ll7/kidinnuaitools.git

# Navigate to project directory
cd kidinnuaitools

# Install dependencies
npm install
# or
bun install

# Start development server
npm run dev
# or
bun dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
# or
bun run build
```

Contributions are welcome! If you'd like to:
